# 演讲比赛流程管理系统
## 演讲比赛程序需求
### 比赛规则
 * 学校举行一场演讲比赛，共有**12人**参加。**比赛共两轮**，第一轮为淘汰赛，第二轮为决赛  
 * 每名选手都有对应的**编号**，如10001~10012  
 * 比赛方式：**分组比赛，每组6人**  
 * 第一轮分为两个小组，整体按照选手编号进行**抽签**后顺序演讲  
 * 十个评委分别给每名选手打分，去除最高分和最低分，求得平均分为本轮选手的成绩  
 * 当小组演讲完后，淘汰组内排名最后的三个选手，**前三名晋级**，进入下一轮的比赛  
 * 第二轮决赛，**前三名胜出**  
 * 每轮比赛过后需要**显示晋级选手的信息**
